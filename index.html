Projects

testR

testR is R test generation framework (also available as a package) for unit tests generation from source code, test execution and translation, and filtering of test cases based on C and R code coverage. Developed as a part of my research assistanship at S3 lab in Purdue university under supervision of Jan Vitek. Presented on useR 2014(slides)

Pander

Pander is an R package is to provide a minimal and easy tool for rendering R objects into Pandoc's markdown. I worked on it as part of my proposal for Google Summer of Code 2014 and Google Summer of Code 2015 under supervision of Gergely Daroczi

LMS-Truffle

A research project that explores the results of combination of generative programming approach (Lightweight modular staging) with self-modidying AST Interpreters (Truffle Framework). This project was done as part of Metaprogramming and Program Generation course under supervision of Prof. Rompf and will be presented at ECOOP's Truffle Workshop
